# Homepage
아이르 필라테스 &amp; PT 의 공식 홈페이지를 위한 저장소입니다.

## 기술 스택
- Vue.js (v3.2.39)
- Node.js (v16.17.0)
- NPM (v8.15.0)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
